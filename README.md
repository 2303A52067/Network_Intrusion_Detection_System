# Network Intrusion Detection System using Deep Learning

This project implements a Deep Learning-based Intrusion Detection System (IDS) using LSTM and Bidirectional LSTM (BiLSTM) architectures to classify network traffic as normal or anomalous.

The system was trained on a network traffic dataset containing 25,000+ connections and 41 features, enabling the detection of malicious activities and potential cybersecurity threats.

## Technologies Used
- Python
- TensorFlow
- Keras
- Pandas
- NumPy
- Scikit-Learn

## Models Implemented
- LSTM
- Bidirectional LSTM (BiLSTM)

## Results
- Test Accuracy: ~87.6%
- Effective anomaly detection for network security applications
